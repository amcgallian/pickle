# Ocean Carbon Chemistry

A collection of geochemical models I built for coursework, put together in one place so they're easier to explore as a set rather than scattered across separate notebooks.

The models all build on a shared CO₂ system solver and cover a few different angles on how carbon moves through the ocean, alkalinity enhancement, transient CO₂ uptake, and mineral weathering kinetics.

There's an [interactive version](https://andrewmcgallian.github.io/pickle/) that runs everything in the browser so you can play with the parameters without touching any code.

## Models

- **CO₂ System Solver** — equilibrium carbonate chemistry via bisection
- **OAE Mixing Model** — surface ocean response to alkalinity and carbon addition
- **Ocean Box CO₂ Uptake** — closed air-ocean column responding to an atmospheric CO₂ spike
- **Mineral Dissolution Kinetics** — pH-dependent weathering of CaO and CaCO₃ in soil water
